About Me

Hey! I'm May, a game programming student at Staffordshire University, and I've been making games since I was 12. My main interests are in rendering technology and engine tools ♥
I'm currently working on a graphics and physics engine in DirectX11 and networking in C# and .NET, and self-teaching Unreal Engine 5 for more design oriented personal projects.

Here you can see some of the projects I have worked on. I love taking on new challenges and persevering through difficult technical goals, my dream is being on the stage at GDC one day!

Student Projects:

Pacman

Recreating the arcade classic in C++ with AI pathfinding, file reading and 2D physics collisions Find the repo here!

3D Game Engine

A 3D rendering engine made in OpenGL 3.3 with Camera and Input classes, file loading and decoupling, Blinn-Phong and Material shading.

Super Mario Remixed!

2D game engine built from scratch in the SDL2 framework with a Mario-themed platformer built on top! Sound programming with SDL Mixer, 2D physics and AI, Event handler class.

Feudal Noir

A 3D scene themed around feudal Japan framed in the 20s Film Noir aesthetic. Modelled in 3DS Max, Textured in Substance Painter and Composed in Unreal Engine for. FMP for City of Wolverhampton College
